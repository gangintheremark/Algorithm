> - [2774 별 찍기 - 10](https://www.acmicpc.net/problem/2447)

알고리즘 중에 제일 약한 부분이 재귀 파트여서 문제를 보고 
순환을 이용해야겠다. 라는 생각이 바로 들지 않았고 계속 생각하니 어떤 구조로 순환이 되는지 이해하여 진짜 3시간만에 풀었다...<br>
 1. 공백이 시작되는 좌표에서 몇 칸을 공백으로 채워야 하는지 -> erase 함수 <br>
 2. 공백이 시작되는 좌표를 어떻게 구하는지 -> draw 함수

처음엔 공백을 지우는 칸이 큰 것 부터 차례로 지워내려 가는걸 생각했는데 잘 안되어 한 칸 부터 지우는 것으로 바꿔 생각했더니 쉽게 풀렸다.
